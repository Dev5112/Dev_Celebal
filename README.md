# Shopping Dataset Analysis

## 📌 Project Overview

This project focuses on performing **Basic Data Exploration, Data Cleaning, Feature Engineering, and Exploratory Data Analysis (EDA)** on a shopping product dataset using Python.

The main objective of this project is to understand product-level information, clean raw data, create useful features, visualize patterns, and generate meaningful business insights.

---

## 🎯 Objective

- Learn basic data handling using Python and Pandas
- Load and explore a CSV dataset
- Clean missing and duplicate data
- Perform feature engineering
- Analyze product pricing, ratings, and categories
- Create visualizations for better understanding
- Save the cleaned dataset

---

## 📂 Dataset

Dataset Used:

`combined_dataset.csv`

The dataset contains shopping product details including:

- Product ID
- Product Title
- Product Description
- Rating
- Ratings Count
- Initial Price
- Final Price
- Discount
- Category
- Seller Information
- Product Details
